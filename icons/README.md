# Extension Icons

This folder should contain the extension icons in the following sizes:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

## Creating Icons

You can create simple placeholder icons using any image editor, or use online tools like:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

For a quick solution, you can create simple colored squares with the text "TR" (Tab Renamer) centered.

## Using SVG (Alternative)

If you want to use SVG instead, you'll need to convert to PNG format since the manifest requires PNG files for Edge extensions.
